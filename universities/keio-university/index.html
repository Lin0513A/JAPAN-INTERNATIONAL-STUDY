慶應義塾大学

最終確認日: 2026-07-10。このページはISJの募集要項DBから生成しています。正式出願前は必ず大学公式の最新募集要項を確認してください。

出願資格

文学部・経済学部・法学部・商学部・理工学部の外国人留学生対象入学試験は、12年以上の学校教育課程を修めた者または2027年3月31日までに修める見込みの者等で、日本以外の国において海外の学校教育制度に基づく中学校および高等学校の全期間の課程を修了した者または修了見込みの者、かつ当該国における大学入学資格を有する者または取得見込みの者が対象。国籍や在留資格による制限はない。

EJU科目

  • 文学部: 日本語 + 総合科目
  • 経済学部: 日本語 + 総合科目 + 数学コース1または数学コース2
  • 法学部: 日本語 + 総合科目 + 数学コース1
  • 商学部: 日本語 + 総合科目 + 数学コース1または数学コース2
  • 理工学部: 日本語 + 数学コース2 + 理科（物理・化学）

JLPT・英語

JLPT: 2027年度募集要項と公式ページ上で、JLPTの必須提出や級指定は確認できない。日本語能力の判定は学部別にEJU日本語の受験指定で行われている。

英語: 文学部・経済学部・法学部・商学部・理工学部はいずれも、TOEFL iBTまたはIELTS Academic ModuleのOfficial Score提出が必要。詳細な有効期限や送付方法は募集要項20-21ページを要確認。

最低成績・参考帯

2027年度募集要項と公式ページ上で、大学全体の共通EJU最低点・合格最低点は明記されていない。学部ごとに必須EJU科目が指定され、文学部・経済学部・法学部・商学部・理工学部はいずれもTOEFL iBTまたはIELTS Academic ModuleのOfficial Score提出が必要。
