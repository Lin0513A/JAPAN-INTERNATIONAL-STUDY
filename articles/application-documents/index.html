材料准备

志望理由书・募集要項・截止时间怎么读

每所学校都不一样，但确认顺序可以固定。先读对，再开始准备。

募集要項先看这7项

  1. 出愿资格：学历、在留、海外学校年数、国籍等条件。
  2. EJU科目：日本语、総合科目、数学コース、理科科目组合。
  3. 英语/JLPT：是否必须提交，成绩有效期，是否可替代。
  4. 出愿期间：Web登録、支付、邮寄必着往往不是同一天。
  5. 提出材料：成绩单、毕业证明、护照、照片、经费材料、志望理由书。
  6. 校内考/面试：考试日、地点、线上/线下、口头试问范围。
  7. 合格后手续：入学金、在留申请、宿舍和奖学金截止。

志望理由书的核心

不要只写“喜欢日本”“喜欢这个学校”。建议按：过去经历 → 想学的问题 → 为什么这个学部 → 入学后计划 → 将来目标。每段都要和学校课程、ゼミ、学部特色对应。

最后检查

出愿前把PDF文件名、年度、学部、方式、截止时间再看一遍。很多错误不是能力问题，而是读错了年度或方式。
